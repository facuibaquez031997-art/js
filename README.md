## Temas de la Semana

- Variables: `var`, `let`, `const` — diferencias y cuándo usar cada una
- Memoria RAM, Garbage Collection y **Hoisting**
- Tipos de datos primitivos: `string`, `number`, `boolean`, `null`, `undefined`, `NaN`
- `typeof` — detectar el tipo de un valor
- Truthy y Falsy — los 6 valores falsy y todo lo demás
- Operadores: aritméticos, comparación estricta (`===`) vs débil (`==`), lógicos
- Control de flujo: `if`, `else if`, `else`, `switch`
- Bucles: `while`, `for` (y el peligro del bucle infinito)
