# 🥋 Gimnasio de Lógica: Las 40 Katas

¡Bienvenidos al gimnasio! Aquí van a entrenar su memoria muscular con JavaScript. Hemos configurado un sistema de **Tests Automatizados** (como usan las empresas reales) para que la computadora les corrija la tarea en tiempo real.

## 🚀 ¿Cómo empezar?

1. Abre tu terminal asegurándote de estar dentro de esta carpeta (`04-Tarea-40-Katas`).
2. Instala el motor de corrección escribiendo: `npm install`
3. Arranca el evaluador escribiendo: `npm run test`

La consola se quedará abierta. A medida que escribas tu código en los archivos `.js` y guardes (`Ctrl + S`), los tests se ejecutarán solos y te dirán si tu código está bien (Verde ✅) o mal (Rojo ❌).

> **⚠️ REGLA DE ORO:** ¡NO toques los archivos que terminan en `.test.js`! Solo debes escribir tu código dentro de las funciones vacías en los archivos normales.

## 📁 Estructura de archivos

| Archivo                   | Katas   | Tema                                |
| ------------------------- | ------- | ----------------------------------- |
| `01-variables-y-tipos.js` | 1 — 10  | Variables, tipos, strings, números  |
| `02-condicionales.js`     | 11 — 20 | if/else, switch, operadores lógicos |
| `03-bucles-basicos.js`    | 21 — 30 | for, while, arrays                  |
| `04-logica-mixta.js`      | 31 — 40 | Todo combinado                      |

## 💡 Pistas clave

- Cada función debe usar `return` para devolver el resultado. **No uses `console.log()`**.
- Si el test dice ❌, leé el mensaje de error: te dice exactamente qué esperaba recibir y qué recibió.
- Completá los archivos en orden, de arriba hacia abajo.
